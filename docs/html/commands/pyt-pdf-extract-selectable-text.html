pyt-pdf-extract-selectable-text

Extracts selectable text from one PDF using a lightweight parser.

Use when:

  • The PDF already has a text layer.
  • OCR is not needed.

Writes:

  • One UTF-8 .txt file.

Dependencies:

  • .[pdf]
